# cn-coinpusher-resource

本仓库用于归档和发布多个独立 HTML 页面。这些页面各自承载不同的功能或信息，彼此之间无强关联，也不指向任何特定域名或网站。

## 项目简介

`cn-coinpusher-resource` 是一个静态资源归档仓库。仓库中的 HTML 文件均为独立页面，可直接在浏览器中打开运行，无需额外后端支持。

这些页面可能包含工具、演示、信息展示或交互原型等内容，统一存放于此以便分发和引用。

## 目录说明

```
/
├── index.html          # 仓库入口页面（如存在）
├── pages/              # 存放各独立 HTML 页面的目录
│   ├── page-a.html
│   ├── page-b.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下每个 `.html` 文件均为一个独立页面。
- `assets/` 目录存放页面共享或引用的静态资源。
- 具体文件用途请参考各页面内的说明或注释。

## 页面归档说明

- 所有页面均为纯前端实现，无后端依赖。
- 页面之间互不干扰，可单独部署或使用。
- 部分页面可能引用 `assets/` 中的公共资源，请确保路径正确。
- 归档内容可能不定期更新或清理，请以仓库当前状态为准。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的任意 `.html` 文件。
3. 或部署到任意静态托管服务（如 GitHub Pages 等）进行访问。
